# Exemples de code des forums MSDN francophones
Ce repository contient des exemples de code sur mes participations concernant les forums
[MSDN francophones](https://social.msdn.microsoft.com/Forums/fr-FR/home).

Pour retrouver l'exemple de code concerné par le fil de discussion, il suffit d'écrire la requête suivante et d'ajouter
l'identifiant du fil de la discussion (non du répertoire qui représente un GUID).

````
https://social.msdn.microsoft.com/Forums/fr-FR/<Identifiant>/
````

Par exemple:

[https://social.msdn.microsoft.com/Forums/fr-FR/b8c217db-ede2-4676-857c-1ec01327e1b3/](https://social.msdn.microsoft.com/Forums/fr-FR/b8c217db-ede2-4676-857c-1ec01327e1b3/)
